# 原理说明


## 编译原理
-----------------------------
源程序 -> 词法分析 -> 语法分析 -> 语义分析 -> 生成中间代码 -> 代码优化 -> 目标程序生成


## 文法
-----------------------------


## 词法分析
-----------------------------
- 输入： 源程序
- 输出： 二元组(种别码，标识)

## 语法分析
-----------------------------
- 输入： 二元组
- 输出： 抽象语法树
- 作用： 识别语法错误，生成维护符号表

## 思路
-----------------------------
- 是否是分隔符，是，就获取单词
- 判断单词是否是关键字
- 判断是否是常量
